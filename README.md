About

Curiofy is a small front-end project focused on delivering a polished UI experience with modern HTML, CSS, and vanilla JavaScript. It’s ideal as a learning resource, a starter template, or a demo of interactive components.

Features

Responsive layout (desktop / mobile)
Interactive UI components and animations
Lightweight: no framework required (vanilla JS)
Easy to extend and customize
Tech stack

HTML
CSS (including responsive and animation styles)
JavaScript (vanilla, DOM-based)
Demo

Live demo: https://lorenasferreira.github.io/curiofy/index.html

Installation

To run locally:

Clone the repo git clone https://github.com/lorenasferreira/curiofy.git
Open the project folder cd curiofy
Open index.html in your browser
Double-click index.html, or
Node (live-server): npx live-server
Note: If the project includes a package.json and a dev server, you can run: npm install npm start

Usage

Navigate the UI to explore components and interactions.
Modify HTML/CSS/JS files in the src (or root) folder to customize.
Add new components under a components/ or src/ directory as needed.
Project structure

index.html — main entry
css/ or styles/ — stylesheets
js/  — JavaScript files
assets/ — images
README.md — this file
